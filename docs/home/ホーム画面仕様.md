# ホーム画面仕様書

## 概要
トヨタ車両管理アプリケーションのホーム画面仕様書です。Android、iOS共通で利用できる仕様として記載しています。

## 画面構成

### 1. ステータスバー
**位置**: 画面最上部
**高さ**: 44pt (iOS) / 24dp (Android)
**背景色**: 透明
**内容**:
- 左側: 時刻表示「14:36」
- 右側: システムアイコン（通信状況、バッテリー残量）

### 2. アプリヘッダー
**位置**: ステータスバーの下
**高さ**: 56pt (iOS) / 56dp (Android)
**背景色**: ダークグレー (#1C1C1E)
**内容**:

- 中央: 車種選択ボタン「ハリアー ハイブリッド」+ 右向き矢印アイコン
- 右側: ハンバーガーメニューアイコン（3本線）

### 3. 車両画像表示エリア

## ユーザーインタラクション

### タップ可能要素
1. **車種選択ボタン**: 車種選択画面へ遷移
2. **ハンバーガーメニュー**: サイドメニュー表示
3. **画像ギャラリーアイコン**: 車両画像ギャラリー表示
4. **車両情報ボタン**: 車両詳細情報画面へ遷移
5. **警告メッセージ**: オートアラーム設定画面へ遷移
6. **ハザードボタン**: ハザードランプのリモート制御
7. **エアコンボタン**: エアコンのリモート制御
8. **カーファインダーボタン**: 車両検索機能の実行
9. **お知らせタブ**: お知らせ画面へ遷移

### 画面遷移
- 車種選択 → 車種選択画面
- 車両情報 → 車両詳細情報画面
- 警告メッセージ → オートアラーム設定画面
- お知らせタブ → お知らせ画面

## アクセシビリティ対応

### スクリーンリーダー対応
- すべてのボタンに適切なアクセシビリティラベルを設定
- 車両ステータス情報に数値の読み上げ対応
- 警告メッセージの優先度設定

### 視覚的アクセシビリティ
- 十分なコントラスト比（4.5:1以上）を確保
- タップターゲットサイズ（44pt/44dp以上）を維持
- 色だけでなくアイコンでも情報を伝達

## レスポンシブ対応

### 画面サイズ対応
- 最小幅: 320pt/320dp
- 最大幅: 414pt/414dp
- 縦横比の維持

### デバイス固有対応
- **iOS固有**: セーフエリア対応、ホームインジケーター対応
- **Android固有**: ナビゲーションバー対応、ステータスバー対応

## プラットフォーム固有実装

### iOS固有
- セーフエリアの考慮
- ホームインジケーターの表示
- ステータスバーの高さ（44pt）

### Android固有
- ナビゲーションバーの考慮
- ステータスバーの高さ（24dp）
- マテリアルデザインガイドラインの適用

## 注意事項
- ダークテーマ専用のデザイン
- 車両画像は実際の車両データに基づいて動的に変更
- ステータス情報はリアルタイムで更新
- ネットワーク接続が必要な機能のオフライン対応
