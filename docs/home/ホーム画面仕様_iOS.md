# ホーム画面仕様書

## 概要
トヨタ ハリアー ハイブリッド車のスマートフォン向けホーム画面UIの詳細仕様

## 画面構成

### 1. ヘッダー部分（上部）
**位置**: 画面最上部
**高さ**: 約60px

#### 1.1 ステータスバー
- **時刻表示**: 左上に「14:36」
- **通信状況**: 電波強度アイコン（4本）
- **WiFi**: WiFiアイコン
- **バッテリー**: バッテリー残量アイコン（約80%）

#### 1.2 タイトルバー
- **背景色**: ダークグレー（#2C2C2E）
- **左側**: トヨタロゴ（赤色、約30x30px）
- **中央**: 「ハリアー ハイブリッド」テキスト（白色、フォントサイズ16px）
- **右側**: ハンバーガーメニューアイコン（3本線）

### 2. メイン車両表示エリア
**位置**: ヘッダー下から画面中央
**高さ**: 約250px
**背景**: ダークグレーのグラデーション

#### 2.1 車両画像
- **位置**: エリア中央
- **サイズ**: 幅約300px、高さ約150px
- **車種**: 黒色のハリアー ハイブリッド
- **角度**: 斜め前方から撮影したアングル

#### 2.2 車両情報ボタン
- **位置**: 車両画像右下
- **アイコン**: イメージ画像マークの小さいボタン
- **サイズ**: 約30x30px
- **背景**: 半透明の暗色

### 3. 車両ステータス情報エリア
**位置**: 車両画像下部
**レイアウト**: 横並び3列

#### 3.1 燃費情報（左列）
- **ラベル**: 「航続可能距離」（薄いグレー、フォントサイズ12px）
- **数値**: 「146km」（白色、フォントサイズ18px、太字）

#### 3.2 給油情報（中央列）
- **アイコン**: 給油ポンプのアイコン（青色）
- **数値**: 「23」（白色、フォントサイズ18px、太字）

#### 3.3 総走行距離（右列）
- **ラベル**: 「総走行距離」（薄いグレー、フォントサイズ12px）
- **数値**: 「6,395km」（白色、フォントサイズ18px、太字）

### 4. 車の情報セクション
**位置**: ステータス情報下
**高さ**: 約40px

#### 4.1 情報ボタン
- **位置**: 中央
- **アイコン**: 情報マーク（ⓘ）
- **テキスト**: 「クルマの情報」（白色、フォントサイズ14px）
- **背景**: 透明
- **ボーダー**: なし

### 5. 警告・通知エリア
**位置**: 車の情報セクション下
**高さ**: 約50px

#### 5.1 オートアラーム警告
- **背景色**: 暗いオレンジ/黄色（#FF8C00）
- **アイコン**: 警告三角マーク（左側）

- **矢印**: 右向き矢印（右端）

### 6. 操作ボタンエリア
**位置**: 警告エリア下
**レイアウト**: 横並び3つの円形ボタン
**ボタン間隔**: 均等配置

#### 6.1 ハザードボタン（左）

#### 6.2 エアコンボタン（中央）

#### 6.3 カーファインダーボタン（右）

- **背景色**: ダークグレー（#3C3C3E）

#### 7.2 サービスアイコン（下部）
**レイアウト**: 横並び2つのアイコン

##### 7.2.2 お知らせアイコン（右）

- **ラベル**: 「お知らせ」（下部、白色、フォントサイズ12px）

### 8. ナビゲーション（最下部）

## インタラクション仕様

### タップ可能要素
1. ハンバーガーメニュー
2. 車両画像（詳細表示）
3. 車の情報ボタン
4. オートアラーム警告
5. 3つの操作ボタン（ハザード、エアコン、カーファインダー）
6. リモートサービスアイコン

### フィードバック
- タップ時: 軽い振動フィードバック
- ボタン押下時: 背景色の明度変更（0.2秒）

## レスポンシブ対応

### iPhone SE (375x667)
- 全体を90%スケール
- ボタンサイズを70pxに調整

### iPhone Pro Max (414x896)
- デフォルトサイズを維持
- 余白を適切に配置

### iPad対応
- 横向き表示時は2カラムレイアウトに変更
- 左側: 車両情報、右側: 操作ボタン群

## アクセシビリティ

### VoiceOver対応
- 全てのボタンに適切なアクセシビリティラベル設定
- 車両ステータス情報を音声で読み上げ

### ダークモード
- 既にダークテーマで設計済み
- ライトモード切り替え時は背景色を#F2F2F7に変更

## 技術仕様

### 推奨実装フレームワーク
- **iOS**: SwiftUI
- **Android**: Compose

### 必要なAPI連携
1. 車両ステータス取得API
2. リモート操作API（ハザード、エアコン等）
3. 位置情報API（カーファインダー用）
4. 通知API（お知らせ用）

### パフォーマンス要件
- 初期表示: 2秒以内
- 車両ステータス更新: 1秒以内
- ボタン応答: 0.3秒以内
